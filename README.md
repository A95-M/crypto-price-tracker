# Crypto Price Tracker 💰

![Screenshot](images/screenshot.png)

A clean and responsive cryptocurrency price tracker that fetches real-time data from CoinGecko API. Includes search, sorting, auto-refresh, Top Gainer badge, and responsive layout — perfect for a Junior Frontend portfolio.

---

## Features ✨

- **Real-time data** from CoinGecko API
- **Search** for cryptocurrencies by name
- **Sort** by price (High → Low, Low → High)
- **24h Price Change** with positive/negative color coding
- **Top Gainer Badge 🔥** highlights the coin with the highest 24h gain
- **Auto-refresh** every 60 seconds
- **Responsive layout** for Desktop, Tablet, and Mobile
- **Simple footer** with copyright & GitHub link

---

## Tech Stack 🛠️

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- CoinGecko API
